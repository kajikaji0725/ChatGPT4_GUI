# 起動方法 poetryいれてね
- poetry install
- poetry run python main.py
  - 実行する前に，main.pyがあるディレクトリに.envファイルを作ってください
    - 中身は，CHATGPT4_APIKEY=???としてください．(???のところに，各自のAPIキーを入れてください)

# 実行形式に変更
- poetry run pyinstaller main.py
- 結果は生成されたdistフォルダの中に入ってます
  - 引数を指定すると，いろいろと便利かもしれませんが，調べていないのでわかりません....
- 動かす際には，dist/mainの中に，.envを作って，main.pyをクリックしてください

# アプリの説明
## ホーム画面
<img width="951" alt="image" src="https://github.com/kajikaji0725/ChatGPT4_GUI/assets/75021542/2f5b1912-8162-4a86-bb30-416a226ea41f">  

- アプリ起動時は，このような画面が出てきます．
- 下のテキストフィールドに，質問事項を入力して，右の送信ボタンを押すとChatGPTに質問することができます．
  - ここで，間違ったAPIキーを.envファイルに保存すると，エラーが出てきます．

<img width="122" alt="image" src="https://github.com/kajikaji0725/ChatGPT4_GUI/assets/75021542/8546da23-a562-44df-bf0b-8556afa618d1">  

- これらはそれぞれ，「会話履歴の削除」，「会話履歴のダウンロード」，「ホーム画面に戻る」，「設定画面に遷移する」ボタンです．
  - 「会話履歴の削除」
    - これは，ChatGPTと会話した履歴を削除するボタンです．
    - 削除すると，ホーム画面上に表示されるChatGPTの会話履歴表示と，質問を送る際に履歴を含めて送る場合の履歴，この両方が初期化されます．
  - 「会話履歴のダウンロード」
    - これは，ChatGPTと会話した履歴をダウンロードするボタンです．
    - ダウンロードする場所はユーザが選択することができます．
    - フォーマットはtxtです
  - 「ホーム画面に戻る」
    - これは，設定画面に遷移した後に，ホーム画面に戻るボタンです
  - 「設定画面に遷移する」
    - これは，設定画面に遷移するボタンです
## 設定画面
<img width="951" alt="image" src="https://github.com/kajikaji0725/ChatGPT4_GUI/assets/75021542/4dbe10b3-2dcc-4f4a-aafa-31b4fd8b6169">  

- 設定画面は，このような画面が出てきます．
- 設定項目として，「strem」，「is_use_history」，「ChatGPT_mode」，「初期化」があります．
  - 「strem」
    - これは，ChatGPTからの応答形態を表しています．
    - stremがTrueであれば，返答形式が，Web版のように文字ごとに更新されます．
    - 初期設定ではTrueです
  - 「is_use_history」
    - これは，ChatGPTに質問する際に，これまでの履歴を送信するか否かを表しています．
    - Trueにすると，これまでの会話履歴を含んだメッセージを送信します．
    - 初期設定ではFalseになっています．
      - 理由として，履歴を含めるとトークン数が多くなるため，一応Falseにしています．
      - 個人の判断でお願いします
  - 「ChatGPT_mode」
    - これは，使用するmodelを設定します．
    - 基本的には，gpt-3.5-turboを使用します．
      - gpt-4を使用する際には，なんか申請する必要があるそうなので，そこは各自でお願いします．
      - 申請していないのに，gpt-4を選択して質問すると，エラーが出てきます．
    - 初期設定ではgpt-3.5-turboになっています
  - 「初期化」
    - これは，会話履歴，各種設定を初期化します．
    - ホーム画面のデリートボタンは，会話履歴の初期化だけですが，ここでの初期化は，会話履歴と各種設定を初期化します．
